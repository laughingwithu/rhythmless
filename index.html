rhythmless

Vertical Rhythm for LESS

rhythmless provides an easy way to maintain a vertical rhythm in LESS/CSS. It's a lightweight mixin bundle that doesn't make any assumptions about your code.

Features

  • Semantic - use the mixins with any class or element
  • Compatible - tightly namespaced and compatible with CSS reset code
  • Easy - put away your calculator, no need for pesky arithmatic
  • Pure - generates pure CSS, no JavaScript
  • Flexible - specify your own rhythmic unit and apply it to the selectors you want

Code

// import the mixins
@import "rhythm";

// set the rhythmic unit
@rhythmic-unit: 20px;
h1 {
    // set the text size to 40px
    .rhythm-text(40px);
    // add a space above and below
    .rhythm-seperator(both);
}
